# Smart Root Health Monitoring System using Low-Frequency Soil EM Signal

## Project Overview
This project focuses on monitoring soil moisture and root health using Machine Learning techniques. The system analyzes soil and environmental parameters to classify soil moisture conditions and support better agricultural decision-making.

## Objectives
- Monitor soil conditions using sensor data.
- Predict soil moisture using Machine Learning models.
- Compare the performance of ANN, CNN, LSTM, and RNN models.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras

## Machine Learning Models
- Artificial Neural Network (ANN)
- 1D Convolutional Neural Network (1D CNN)
- Long Short-Term Memory (LSTM)
- Recurrent Neural Network (RNN)

## Dataset
The dataset contains:
- Humidity
- Atmospheric Temperature
- Soil Temperature
- Dew Point
- Soil Moisture

## Future Scope
- Real-time IoT monitoring using ESP32.
- Smart agriculture applications.
- Live dashboard for soil monitoring.
